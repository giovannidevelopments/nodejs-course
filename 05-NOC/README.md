# Proyect NOC
The objective is to create a series of tasks using Clean Architecture with TypeScript

# dev
1. Clone the env.template file to .env
2. Configure the environment variables
3. Run ``` npm install```
4. Start the databases with the command
    ```
    docker compose up -d.
    ```
5. Run ``` npm run dev ```
